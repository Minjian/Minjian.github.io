Projects

Rogas - A declarative framework for network analytics. It not only can provides a high-level declarative query language to formulate analysis queries, but also can unify different graph algorithms within the relational environment for query processing.
STM32_Air_Purifier - Project for an Air Purifier that can detect the PM2.5 level and peculiar smell and thus control the air quality of the room. The air purifier supports remote control by using the iOS or Android application.
Qt_Video_Monitoring - Project for a Video Monitoring System used by Shanghai XianXia High School and Shanghai Experimental Kindergarten.
Tiny4412 Driver Development - Tiny4412 is an embedded development board with the Samsung Exynos 4412 chip. This project is about driver development for several customized modules on the Tiny4412 board, including LED driver, PWM Beeper driver, ADC driver, Touch Screen and LCD driver, WiFi driver(RTL8192CU), Audio driver(WM8960), Ethernet driver(DM9621). Drivers for other modules (e.g. eMMC, RTC, I2C-EEPROM, USB, SD/MMC, FIMC, HDMI) are provided in the official Samsung BSP.
